The repeat() Method

The   repeat()   returns a string with a number of copies of a sring.

The   repeat()   returns a new string.

The   repeat()   does not change the original string.

Examples 

                                                            let text = "Hello, World! ";
                                                            let result = text.repeat(2);

repeat() is not supported in Internet Explorer
