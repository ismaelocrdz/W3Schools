JavaScript Strings Methods 13

String trim()

The   trim()   methods removes whitespaces from both sides of a string.

Example 

                    let text01 = "      Hello, World!      ";
                    let text02 = text01.trim();

String trimStart()

ECMAScript 2019 added the String method   trimStart()   to JavaScript.

The   trimStart()   method works like   trim()   , but removes whitespaces only from the start of a string.

JavaScript String   trimStart()   is supported in all modern browsers since JAnuary 2020.
