Dynamic HTML content

JavaScript can create dynamic HTML content:

Date: Sat Feb 03 2024 17:29:59 GMT-0600 (hora estándar central)

Example 

                    <!DOCTYPE html>
                    <html>
                    <body>                    
                        <script>
                            document.getElementById("demo").innerHTML = "Date : " + Date(); </script>                    
                    </body>
                    </html>
