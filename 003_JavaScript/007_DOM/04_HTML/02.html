JavaScript HTML DOM Changing HTML 02

Example explained:

  • The previous HTML document contains a    <p>    elemnt with    &id="demo01"   
  • We use the HTML DOM to get the element with    id="demo01"   
  • A JavaScript changes the content (    innerHTML    ) of that element to "New Text!"

The next example changes the content of an    <h1>    

                    <!DOCTYPE html>
                    <html>
                    <body>                    
                        <h1 id="id01">Old Heading</h1>                    
                        <script>
                            const element = document.getElementById("id01");
                            element.innerHTML = "New Heading";
                        </script>                    
                    </body>
                    </html>
