The padStart() Method

The   padStart  Method pads a string from the start.

It pads the string with another string (multiple times) until it reaches a given legth.

Example

Pad a string with "0" until it reaches the legth 4: 

                    let text01 = "5";
                    let padded = text01.padStart(4,"0");

Pad a string with "x" until it reaches the legth 5: 

                    let text01 = "7";
                    let padded = text01.padStart(7,"x");
