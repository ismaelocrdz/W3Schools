The   at()   method returns the character at a spiecified index (position) in a string.

The   at()   method is supported in all modern browsers sins March 2022.

Note

The   at()   method is a new addition to JavaScript.

It allows the use of negative indexes while   charAt()   do not.

Now you can use   myString(2).at(-2)   instead of   charAt(myString.length-2)   .
