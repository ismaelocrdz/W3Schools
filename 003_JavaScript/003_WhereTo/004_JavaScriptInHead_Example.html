The <script> Tag (Code)

Javascript in Body 

                    <!DOCTYPE html>
                    <html>
                    <head>
                        <script>
                        function myFunction() {
                        document.getElementById("demo").innerHTML = "Paragraph changed";
                        }
                        </script>
                    </head>
                    <body>
                        <h2>Demo JavaScript in Head</h2>
                        <p id="demo">A Paragraph</p>
                        <button type="button" onclick="myFunction()">
                            Try it
                        </button>
                    </body>
                    </html>
